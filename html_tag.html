목록 만들기

  1. 순서가 있는 목록 ol태그
  2. 리스트 내용은 li 태그 안에 기록.
  3. 순서를 만드는 type 5가지가 있다.
  4. ol 태그 내의 type속성으로 지정.
  5. 값 : 1, A, a, I, i
  6. 시작번호는 start 속성으로 숫자를 적어준다.
  • 순서가 없는 목록은 ul 태그를 사용한다.
  • 숫자나 문자대신에 블릿이 앞에 붙게 된다.
  • 속성은 없고, 사용은 ol 태그와 같이 ul 태그안에 li 태그를 사용해서 목록을 구성한다.
